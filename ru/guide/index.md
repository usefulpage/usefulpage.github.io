# Руководство пользователя

**Useful Tools** — многофункциональный сайт с бесплатными онлайн‑инструментами. Всё работает в браузере, без регистрации.

## Основные возможности

- Генератор паролей
- Генератор случайных чисел
- Генератор фактов
- Калькулятор
- Генератор ников для Minecraft
- Генератор оружия CS2
- Генератор рабочих имён
- Совет дня
- Демо-переводчик
- **Игра «Угадай число» на Python** (через Pyodide)
- **30+ Minecraft инструментов** (скины, зачарования, баннеры, координаты, tellraw, MOTD и др.)
