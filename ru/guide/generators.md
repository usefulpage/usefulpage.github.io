# Генераторы и калькуляторы

На сайте представлен широкий набор инструментов, которые работают прямо в браузере. Все они бесплатны, не требуют регистрации и не оставляют ваши данные на сервере (если только вы сами не согласились на расширенное логирование).

## Генератор паролей

Создаёт случайные пароли заданной длины.

- Можно включить заглавные буквы, цифры и спецсимволы.
- Длина от 4 до 64 символов.
- Результат можно скопировать в буфер обмена кнопкой «📋».

## Генератор случайных чисел

Выдаёт целое число в заданном вами диапазоне.

- Поля «Мин.» и «Макс.» принимают любые целые числа.
- Если минимум больше максимума, они автоматически меняются местами.

## Генератор фактов

Показывает случайный удивительный факт из коллекции (около 10–20 фактов на каждый язык).

- Факты переведены на все доступные языки интерфейса.
- При каждом нажатии кнопки «Новый факт» загружается новый факт (без обращения к серверу).

## Калькулятор

Обычный арифметический калькулятор с кнопками.

- Поддерживает сложение, вычитание, умножение, деление.
- Есть кнопка «C» для очистки.

## Генератор ников для Minecraft

Составляет случайный ник из приставки (`Miner`, `Craft`, `Diamond` и т.д.) и суффикса (`Pro`, `Gamer`, `Lord` и т.д.), плюс случайное число от 0 до 999.

## Генератор оружия для CS2

Выдаёт случайное оружие из списка (AK-47, M4A4, AWP, Deagle, USP-S, Glock, SSG 08 и ещё 15+ видов).

## Генератор рабочих имён

Создаёт креативное профессиональное имя (например, «Creative Designer», «Strategic Developer»). Может пригодиться для резюме или псевдонима.

## Совет дня

Показывает случайный полезный совет (про здоровье, продуктивность или счастье). Коллекция советов переведена на все языки.

## Демо-переводчик

Поддерживает перевод только четырёх слов: `hello`, `world`, `thank you`, `good morning`. Перевод выполняется по встроенному словарю без обращения к внешним API. **Это демонстрация**, а не полноценный сервис.

## Python игра «Угадай число»

Игра, в которой компьютер загадывает число от 1 до 100, а у вас есть 7 попыток его угадать.

- **Важно:** вся логика написана на Python и выполняется прямо в браузере через Pyodide (WebAssembly).
- Первый запуск может занять 5–10 секунд, так как браузер загружает интерпретатор Python (около 7 МБ).
- Повторные запуски быстрее, так как код кэшируется.

## Minecraft инструменты (30+ утилит)

Выделены в отдельную страницу (`/minecraft-tools`). Включают:

- Скачивание скина по нику (через Mojang API).
- Калькулятор зачарований.
- Галерею баннеров и щитов.
- Калькулятор координат.
- Генератор мобов (`/summon`).
- Таблицы дропа.
- Редактор `/tellraw` и JSON-текста.
- Генератор MOTD для сервера.
- И многое другое.

Полный список доступен на самой странице.
