# Админ-панель

Админ-панель доступна по адресу `https://useful.page.gd/admin.php`. Вход защищён паролем, который задаётся в файле `admin-config.php` (пароль хранится в виде хэша).

## Возможности

### Дашборд

Показывает три карточки:
- Уникальные посетители (количество уникальных `user_id`).
- Всего действий (сумма всех логов).
- Количество файлов логов.

### Графики и аналитика (в стиле YouTube Studio)

- **Посещаемость за последние 30 дней** (линейный график).
- **Активность по часам** (гистограмма, часовой пояс UTC+3).
- **Топ действий** (горизонтальная гистограмма: PAGE_VIEW, TOOL_USE, LANG_CHANGE и т.д.).
- **География** (топ 5 стран по данным GeoIP).
- **Популярные инструменты** (топ 10 по TOOL_USE).
- **Топ страниц** (по PAGE_VIEW).

### Живая лента последних действий

Обновляется каждые 30 секунд. Показывает время, IP, действие и краткие детали 20 последних записей из самого нового файла лога.

### Фильтры и поиск

- Выбор файла лога (по дате).
- Поиск по всему содержимому строки.
- Фильтр по IP.
- Фильтр по действию.
- Фильтр по дате (начало строки времени).

### Таблица логов

Отображает:
- Время, UserID (обрезанный), IP, порт, действие.
- Детали (обрезано, с всплывающей подсказкой).
- URL (обрезано).
- Геолокация (город, страна).
- Куки и заголовки (кратко, иконки 🍪 и 📡).
- Кнопка **🔍** для просмотра **всех 16 полей** записи в модальном окне.

### Пагинация

По 100 строк на страницу. Фильтры сохраняются при переключении страниц.

### Управление логами

- **Очистить все логи** (кнопка, с подтверждением).
- **Удалить логи старше N дней** (N задаётся пользователем).

### Экспорт в CSV

При выборе файла и нажатии на кнопку «Экспорт CSV» скачивается текущий (отфильтрованный) лог в формате CSV, пригодном для открытия в Excel/Google Sheets.

## Как добавить нового администратора

Измените файл `admin-config.php`:

```php
<?php
$admin_user = 'admin';
$admin_hash = '$2y$10$...'; // хэш пароля, сгенерированный через password_hash()
?>```

### Безопасность
- Файл admin-config.php защищён через .htaccess от прямого доступа.

- Сессия администратора живёт до закрытия браузера.

- Все действия администратора (просмотр логов, очистка, экспорт) не логируются (чтобы не засорять лог).
