# Логирование действий

Сайт собирает обезличенные данные для улучшения сервиса, отладки и анализа популярности инструментов. Вся информация хранится на сервере в папке `/logs/` и доступна только администратору.

## Какие данные собираются

Для каждого запроса (PAGE_VIEW) и каждого пользовательского действия (TOOL_USE, LANG_CHANGE, THEME_CHANGE, BVI_ENABLE/DISABLE, CLIENT_INFO и т.д.) в лог-файл записывается:

- **Время** (Y-m-d H:i:s)
- **UserID** — постоянный идентификатор пользователя (хранится в куке `user_id` 365 дней)
- **IP** и **порт** клиента
- **Действие** (PAGE_VIEW, TOOL_USE и т.д.)
- **Детали** (например, `tool=password_generator`)
- **Метод HTTP** (GET/POST)
- **URL** запроса
- **Referer** (откуда пришёл)
- **User-Agent** (браузер, ОС)
- **Строка запроса** (GET-параметры)
- **Accept-Language**
- **Куки** (все, кроме `user_id`, в виде JSON)
- **Заголовки** (Accept, Accept-Language, Accept-Encoding, Connection, Upgrade-Insecure-Requests, Sec-Fetch-*)
- **Геолокация** (по IP через ip-api.com: город, страна, координаты, провайдер, таймзона)

## Клиентское логирование (JS)

Помимо серверного, на сайте работает скрипт `client-logger.js`, который собирает:

- **Информацию об устройстве**: разрешение экрана, цветность, соотношение пикселей, платформу, язык, версию браузера, количество ядер, память, плагины.
- **Тип соединения**: 4G/3G/WiFi, скорость, RTT.
- **Отпечаток браузера** (fingerprint) на основе комбинации параметров.
- **Cookies и localStorage** (содержимое).
- **Клики** (координаты, время между кликами).
- **Глубину скролла** (в процентах).
- **Положение мыши** (с троттлингом 3 секунды).
- **Время на странице** (отправляется при закрытии вкладки).

Все клиентские данные отправляются через AJAX на `/ajax-logger.php` и записываются в те же файлы логов с действиями `CLIENT_INFO`, `FINGERPRINT`, `USER_CLICK`, `SCROLL_DEPTH`, `MOUSE_POS`, `PAGE_DURATION`, `COOKIES`, `LOCALSTORAGE`.

## Доступ к логам

- **Через админ-панель** (`/admin.php`): графики, фильтры, экспорт в CSV.
- **Через скрипт `view-logs.php`** (требует отдельного пароля) – простой просмотр всех записей.
- **По FTP**: папка `/logs/` защищена через `.htaccess` (доступ запрещён).

## Юридическая сторона

Сбор описанных данных осуществляется **только после получения явного согласия пользователя** через плашку cookie-consent (пункт «Аналитические cookies» и «Агрессивное логирование»). Пользователь может отключить сбор в любой момент в настройках плашки.
