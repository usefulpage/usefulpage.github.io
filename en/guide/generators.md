# Generators and Calculators

All tools work directly in your browser. They are free, require no registration, and do not store your data on the server (unless you voluntarily agree to extended logging).

## Password Generator

Creates random passwords of a specified length.

- Options: uppercase letters, digits, special symbols.
- Length: 4–64 characters.
- Copy result with the “📋” button.

## Random Number Generator

Returns a random integer in a range you set.

- Fields “Min” and “Max” accept any integers.
- If Min > Max, they are swapped automatically.

## Random Fact Generator

Shows a random interesting fact from a built‑in collection (10–20 facts per language).

- Facts are translated into all interface languages.
- Each click loads a new fact without a server request.

## Calculator

Basic arithmetic calculator with buttons.

- Supports +, -, *, /.
- “C” button to clear.

## Minecraft Nickname Generator

Generates a random nickname from a prefix (`Miner`, `Craft`, `Diamond`…) and a suffix (`Pro`, `Gamer`, `Lord`…), plus a random number 0–999.

## CS2 Weapon Generator

Outputs a random weapon from a list (AK‑47, M4A4, AWP, Deagle, USP‑S, Glock, SSG 08, and 15+ more).

## Work Name Generator

Creates a creative professional name (e.g., “Creative Designer”, “Strategic Developer”). Useful for a resume or alias.

## Daily Life Tip

Shows a random useful tip (health, productivity, happiness). The tip collection is translated into all languages.

## Demo Translator

Only supports translation of four words: `hello`, `world`, `thank you`, `good morning`. Translation uses a built‑in dictionary without external APIs. **This is a demo**, not a full translation service.

## Python Game “Guess the Number”

The computer picks a number from 1 to 100; you have 7 attempts to guess it.

- **Important:** all logic is written in Python and runs directly in your browser via Pyodide (WebAssembly).
- First launch may take 5–10 seconds (downloads ~7 MB Python interpreter).
- Subsequent launches are faster due to caching.
