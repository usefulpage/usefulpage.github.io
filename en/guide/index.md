# User Guide

**Useful Tools** is a multi‑functional website with free online tools. Everything works in your browser, no registration required.

## Main features

- Password generator
- Random number generator
- Random fact generator
- Calculator
- Minecraft nickname generator
- CS2 weapon generator
- Work name generator
- Daily life tip
- Demo translator
- **“Guess the number” Python game** (via Pyodide)
- **30+ Minecraft tools** (skins, enchantments, banners, coordinates, tellraw, MOTD, etc.)

## Technical highlights

- **12 interface languages** (Russian, English, Ukrainian, Chinese, and 8 variants of Belarusian)
- **Dark / light / system theme**
- **Visually impaired version (BVI)** – increases font size, contrast, removes background images
- **Action logging** – IP, port, geolocation, browser fingerprint, clicks, scroll, cookies
- **Admin panel** – traffic charts, top tools, geography, filters, CSV export
- **PWA** – installable as an app (works even on InfinityFree via blob manifest)
- **Full GDPR compliance** – cookie consent banner for data processing
