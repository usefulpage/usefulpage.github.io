# Admin Panel

The admin panel is available at `https://useful.page.gd/admin.php`. Access is password‑protected; the password is stored as a hash in `admin-config.php`.

## Features

### Dashboard

Three cards:
- Unique visitors (distinct `user_id`).
- Total actions (sum of all logs).
- Number of log files.

### Charts & Analytics (YouTube Studio style)

- **30‑day traffic** (line chart).
- **Hourly activity** (bar chart, UTC+3 time zone).
- **Top actions** (horizontal bar chart: PAGE_VIEW, TOOL_USE, LANG_CHANGE, etc.).
- **Geography** (top 5 countries from GeoIP).
- **Popular tools** (top 10 by TOOL_USE).
- **Top pages** (by PAGE_VIEW).

### Live latest actions feed

Updates every 30 seconds. Shows time, IP, action, and brief details of the last 20 records from the newest log file.

### Filters and search

- Select log file by date.
- Full‑text search across log lines.
- Filter by IP.
- Filter by action.
- Filter by date (start of timestamp).

### Log table

Displays:
- Time, UserID (truncated), IP, port, action.
- Details (truncated, with tooltip).
- URL (truncated).
- Geolocation (city, country).
- Cookies and headers summary (🍪, 📡 icons).
- **🔍 button** to view **all 16 fields** of the record in a modal window.

### Pagination

100 lines per page. Filters persist when switching pages.

### Log management

- **Clear all logs** (confirmation required).
- **Delete logs older than N days** (N user‑defined).

### CSV export

When a file is selected, clicking “Export CSV” downloads the current (filtered) log as a CSV file ready for Excel/Google Sheets.

## How to add a new administrator

Edit `admin-config.php`:

```php
<?php
$admin_user = 'admin';
$admin_hash = '$2y$10$...'; // password hash generated by password_hash()
?>```

### Security
- admin-config.php is protected by .htaccess from direct access.

- Admin session lasts until the browser is closed.

- Admin actions (view logs, clear, export) are not logged to avoid polluting the logs.
